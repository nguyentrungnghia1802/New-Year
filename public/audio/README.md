# Audio Files

Thêm các file âm thanh sau vào thư mục này:

## File cần thiết:

1. **background.mp3** - Nhạc nền chung (nhẹ nhàng, lặp lại)
   - Ví dụ: Nhạc xuân, nhạc Tết truyền thống

2. **fireworks-special.mp3** - Nhạc đặc biệt cho pháo hoa (sôi động)
   - Ví dụ: Nhạc EDM, nhạc festival

3. **click.mp3** - Âm thanh click nút (ngắn, rõ ràng)
   - Ví dụ: Tiếng "pop" nhẹ

4. **flower.mp3** - Âm thanh click vào hoa (nhẹ nhàng)
   - Ví dụ: Tiếng chuông nhỏ, tiếng lá rơi

5. **lixi.mp3** - Âm thanh mở lì xì (vui tươi)
   - Ví dụ: Tiếng pháo nổ nhẹ, tiếng "tada"

6. **firework.mp3** - Âm thanh pháo hoa (sắc nét)
   - Ví dụ: Tiếng pháo nổ "boom"

7. **fortune.mp3** - Âm thanh xem quẻ (trang nghiêm)
   - Ví dụ: Tiếng chuông chùa, tiếng mõ

## Nguồn tải miễn phí:

- **Pixabay**: https://pixabay.com/sound-effects/

## Lưu ý:

- Định dạng: MP3 (khuyến nghị)
- Kích thước: Nhỏ gọn để tối ưu tải trang
- Chất lượng: 128kbps - 192kbps là đủ
- Bản quyền: Đảm bảo sử dụng nhạc free hoặc có license
